# Seriously they don't care!
Over the years I have work on so many projects and with so many technologies.
I've work on some dog shit legacy code bases written in VB and I've also worked with some of the latest and greatest.

The one thing I have found is that these systems is that the users don't give a fuck what it's written in.
They systems provided them with value by giving them a solution to their problem.

Stop following trends build with what you know.

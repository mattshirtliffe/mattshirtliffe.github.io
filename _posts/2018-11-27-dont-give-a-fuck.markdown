# Seriously they don't!
Over the years I have work on so many projects and with so many technologies.
My time as a software developer, I've work on some dog shit legacy code bases written in VB and I've worked with some of the latest and greatest technologies.

The one thing I have found with these systems is that the users don't give a fuck what it written in, they didn't know and didn't care.

These systems provided value by giving them a solution to their problems and the customers paid.

Stop following trends and build with what you know.

You like Python, build using it, you still write your iOS apps in Objective-C so what!

Our goal as developers is to use our skill set to solve problems for our users. So get building!XS
